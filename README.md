# Case Técnico: Analista de Suporte - Dadosfera

Este repositório contém a resolução do desafio técnico para a vaga de Analista de Suporte v2.0 na Dadosfera. O projeto engloba troubleshooting de pipelines, configuração de infraestrutura em Linux, gerenciamento de banco de dados SQL e integração com a plataforma Dadosfera via VPN.

## Tecnologias Utilizadas
* **OS:** Windows
* **Banco de Dados:** PostgreSQL
* **VPN:** OpenVPN
* **Plataforma:** Dadosfera (Catálogo e Pipelines)
* **Linguagem:** SQL

---

##  Estrutura do Desafio

### Step 1 - Troubleshooting
Simulação de atendimento ao cliente sobre falha na importação de dados via Google Sheets. 
* **Solução:** Identificação de causas (permissões, formatação e análise de logs).
* *Resposta completa disponível na pasta: `respostas/`*

### Step 2 - Criação de Banco de Dados
Configuração de um ambiente local para simular a origem dos dados de um cliente.
* **Banco criado:** `igor_gustavo_vaga`
* **Tabela:** `TB_igor_gustavo_vaga`
* **Scripts SQL:** Localizados em `scripts/setup_db.sql`
